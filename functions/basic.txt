function is a set of statements or  block of code that performs some operation or calculations on  the input value.
It will work after invoking it.
syntax of function is like this:
    funciton funcitonName(parameters){
        console.log("hello world! man")
    }

after invoking it like this:
    funcitonName()

here parameters wil behave like local variables into funciton.


if you don't invoke or call the function it will not be read or run.

    2. arrow function (
        example:
            const hello = () =>{
